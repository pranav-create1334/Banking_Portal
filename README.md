🏦 FinGuard – Smart Banking Portal

A modern, secure, and responsive banking web application built using React + Vite. This project simulates real-world banking operations like account management, transactions, and dashboards with a clean UI.

🚀 Features
🔐 User Authentication (Login/Register UI)
💳 Account Dashboard (Balance, Overview)
📊 Transaction History
💸 Fund Transfer UI
📈 Analytics Dashboard (charts & stats)
🌙 Dark/Light Mode
📱 Fully Responsive Design
⚡ Fast performance with Vite
🛠️ Tech Stack
Frontend: React, Vite
Styling:  CSS Modules
State Management: React Hooks / Context API
Icons & UI: Lucide Icons / Custom Components

⚙️ Installation & Setup
# Clone the repo
git clone https://github.com/your-username/bank-frontend.git

# Navigate into project
cd bank-frontend

# Install dependencies
npm install

# Run the project
npm run dev

App will run at:

http://localhost:5173
